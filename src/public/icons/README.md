# PWA Icons

Place your PWA icons here:

- `icon-16x16.png` - Favicon small
- `icon-32x32.png` - Favicon standard
- `icon-192x192.png` - PWA icon Android
- `icon-512x512.png` - PWA icon large
- `apple-touch-icon.png` - iOS icon

## Generate Icons

Use [PWA Asset Generator](https://www.pwabuilder.com/imageGenerator) to generate all required sizes from a single logo.
